# How to use your Android app logo for BYSEL website

1. Locate your Android app logo:
   - Path: android/app/src/main/res/mipmap-xxhdpi/ic_launcher.png

2. Copy the logo file:
   - Copy ic_launcher.png from the above path.

3. Paste into website public directory:
   - Path: website/public/bysel-logo.png
   - Rename the file to bysel-logo.png for consistency.

4. Refresh your website:
   - The logo will now display on the BYSEL website Home page.

---

If you want to use a different resolution, you can choose ic_launcher.png from mipmap-xhdpi, mipmap-xxxhdpi, etc.

This ensures your branding is consistent across Android and web platforms.
