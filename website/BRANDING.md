# BYSEL Branding

Primary Color: #7C4DFF (Purple)
Secondary Color: #00B050 (Green)
Accent Color: #FFB300 (Amber)
Background: #1A1A2E (Dark Blue)
Text: #FFFFFF (White)

Logo: ic_launcher.png (from Android app)

Font: Inter, Roboto, or similar modern sans-serif

Usage:
- Use primary color for headers, buttons, and highlights.
- Secondary color for positive actions and accents.
- Accent color for warnings or highlights.
- Background color for main sections.
- Logo should be placed in the website's public directory as bysel-logo.png.

Replace placeholder logo in public/ with bysel-logo.png for consistent branding.
